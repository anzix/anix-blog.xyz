Wine-tkg - самый пропатченный пользователями Wine

2023-03-03T12:04:11+05:00

linux-gaming
Оглавление
  1. Свежий скомпилированный Wine-tkg
  • Github страница wine-tkg

Процесс установки прост

mkdir ~/Repos && cd ~/Repos
git clone https://github.com/Frogging-Family/wine-tkg-git.git
cd wine-tkg-git/wine-tkg-git/

Перед компиляцией на ваше усмотрение можно править настройки файла customization.cfg чтобы выжать доп-ную производительность. Если лень читать и разбираться но выжать производительность хочется можно обратится на данные ресурсы указанные ниже: ARU (Wine-TKG)

Если готовы тогда начинаем компиляцию (в моём случае с правкой customization.cfg это заняло больше 1 часа)

makepkg -sric

На все вопросы просто тыкаете Enter

#  Свежий скомпилированный Wine-tkg

Идёте на github страницу и выбираете по дате самый последний с отмеченной зелённой галочкой

Опускаетесь вниз до Artifacts и кликаете на "wine-tkg-build" для скачивания

Или скачать его по ссылке

Распаковываете и устанавливаете используя pacman

sudo pacman -U fsync-git-version-x86_64.pkg.tar.zst
