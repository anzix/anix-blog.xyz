!Отображение иконок в терминале

2023-03-29T23:39:35+05:00

linux

Для отображения иконок необходимо юзать шрифты из семейства Nerd. Их очень много, на любой вкус и цвет, и все они имеют поддержку иконок eza/lsd/colorls а также powerlevel10k для вашего терминала.

Примеры шрифтов:

  1. Hack Nerd Font
sudo pacman -S ttf-hack-nerd
  1. JetBrains Mono Nerd Font
sudo pacman -S nerd-fonts-jetbrains-mono
  1. SauceCodePro Nerd Font
sudo pacman -S ttf-sourcecodepro-nerd

Для большего выбора пропатченных шрифтов Nerd смотрите на официальных репозиториях арч или на AUR

Применение шрифта для терминала alacritty:

nano ~/.config/alacritty/alacritty.yml

Раскомментируем строчки а также вводим наш шрифт как показано ниже

font:
 normal:
   family: JetBrains Mono Nerd Font

Рекомендация: Вы можете также дополнительно расписать шрифт JetBrains Mono Nerd Font на других начертаниях (bold, italic, bold italic) в вашем конфиг файле.

Применение шрифта для терминала kitty:

Создать или открыть ~/.config/kitty/kitty.conf и добавить в строчку данный шрифт

font_family JetBrains Mono Nerd Font

Шрифт применился, теперь иконки будут нормально отображаться

Если у вас установлен powerlevel10k после применения шрифта необходимо вызвать мастера настройки приглашения p10k данной командой

p10k configure
